## 스터디 규칙

1. 최소 10분 발표하자
2. 스터디 끝나고 md에다가 정리한거 폴더 만들고 올리기
    - 폴더 이름 규칙 -> 차시_이름_주제
    
3. 주제를 정했으면 페메방에다 알리기
4. 초보자도 이해할 수 있는 발표를 준비하자
5. 발표할 때 집중하기
6. 갑분싸 만들지 않기
7. 커뮤니티가 잘되는 발표 분위기 만들기
8. 발표는 매주 금요일 7교시에 시작
